# 长沙 24 小时特种兵极限吃喝指南

这是一份为大学生和体力极好者设计的“特种兵”行程，主打“只吃不睡”。

## 时间轴规划

### 上午：嗦粉与开胃
*   **08:00 - 肆姐面粉馆**
    *   *必点*：肉饼蒸蛋粉（微辣）。就在坡子街附近，张艺兴推荐过。
*   **09:30 - 岳麓山（仅坐索道）**
    *   *攻略*：不爬山，直接从东门坐滑道下山，体验速度与激情。

### 中午：湘菜洗礼
*   **12:00 - 笨萝卜浏览做饭**
    *   *必点*：酸菜炒粉皮、小炒黄牛肉。
    *   *注意*：排队极恐怖，建议 10:30 就去拿号，过号作废。
*   **14:00 - 茶颜悦色（第 1 杯）**
    *   *推荐*：幽兰拿铁（先吃坚果再喝茶）。长沙五步一家，找人不多的店买。

### 下午：小吃扫荡
*   **16:00 - 黑色经典臭豆腐**
    *   *口感*：灌满汤汁，外焦里嫩。比普通路边摊好吃。
*   **17:00 - 金记糖油粑粑**
    *   *位置*：南门口。长郡中学对面，一块五一个，软糯拉丝。

### 晚上：夜生活与夜宵
*   **20:00 - 杜甫江阁**
    *   *活动*：看湘江夜景，吹江风。如果不进去不用买票，路边拍照更好看。
*   **22:00 - 扬帆夜市**
    *   *目标*：这里是长沙最大的夜市。推荐“小龙虾杯”、“烈火牛肉”。
*   **00:00 - 这里的夜生活才刚开始**
    *   前往解放西路感受“不夜城”氛围，或者去洗脚城（金色印象）过夜省住宿费。
