# 全球热门旅游地常见骗局与防范指南

出门在外，安全第一。以下是经过社区验证的高频骗局，请务必背诵。

## 1. 欧洲篇（巴黎、罗马、巴塞罗那）

### 1.1 手绳党/鸽子食
*   **场景**：在教堂或广场（如米兰大教堂前），有人强行往你手上套彩色绳子，或者往你手里塞玉米喂鸽子。
*   **后果**：一旦你接了，就会有几个壮汉围上来要钱（通常 20-50 欧元）。
*   **对策**：**双手插兜**，冷脸快速通过，不要有眼神接触。

### 1.2 假警察查护照
*   **场景**：身穿便衣自称警察，要求检查你的护照和钱包（理由是检查假币）。
*   **后果**：趁乱抽走你的现金。
*   **对策**：真警察绝不会在街头检查钱包。要求对方带你去最近的警局再检查，对方通常会立刻逃跑。

## 2. 东南亚篇（泰国、越南）

### 2.1 大皇宫“今天不开门”
*   **场景**：你在去大皇宫/卧佛寺的路上，遇到热心的突突车司机或路人，告诉你“今天有皇室活动，大皇宫不开门”，建议带你去另一个“更灵验”的寺庙。
*   **后果**：把你拉到偏远的珠宝店或西装店强行消费。
*   **对策**：**相信官方**。径直走到大皇宫售票处，不要理会路边的搭讪者。

### 2.2 摩托车租赁陷阱
*   **场景**：租车时押护照，还车时店主指着一道旧划痕说是你弄的，索要高额赔偿。
*   **后果**：护照在对方手里，不得不赔钱。
*   **对策**：
    1.  坚决**不抵押护照原件**（可用复印件+押金）。
    2.  提车前围绕车身拍摄 **360度高清视频**，特写所有旧伤痕，并当面发给店主确认。

## 3. 紧急求助方式
*   遇到纠纷保持冷静，优先保护人身安全，财物是次要的。
